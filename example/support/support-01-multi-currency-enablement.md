- **Customer Need**: Acme Corp is expanding to European markets and needs to

(USD, EUR, GBP) based on user location or preference. Currently platform only
supports USD.

## Actions Taken

- Enabled multi-currency feature flag for customer account
- Configured EUR and GBP exchange rate APIs
- Set up automatic currency conversion rules
- Provided documentation on managing currency settings
- Trained customer admin team on currency configuration

USD, EUR, and GBP with automatic conversion based on user's location or manual
selection.
